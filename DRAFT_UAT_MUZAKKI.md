# DRAFT UAT (User Acceptance Testing) - Role: MUZAKKI

Berikut adalah kuesioner pengujian UAT (User Acceptance Testing) untuk peran Muzakki (Donatur) pada Sistem Penyeleksi Mustahik Penerima Zakat. Skala yang digunakan adalah Skala Likert (1-5):
1: Sangat Tidak Setuju, 2: Tidak Setuju, 3: Cukup Setuju, 4: Setuju, 5: Sangat Setuju.

| No | Pernyataan Pengujian (UAT) | 1 | 2 | 3 | 4 | 5 |
|----|---------------------------|---|---|---|---|---|
| 1 | Proses pendaftaran akun Muzakki mudah dipahami dan cepat dilakukan. | | | | | |
| 2 | Halaman login memiliki instruksi yang jelas dan berfungsi dengan baik. | | | | | |
| 3 | Antarmuka formulir pembayaran zakat mudah dimengerti (pemilihan program, nominal, dll). | | | | | |
| 4 | Fitur opsi pengiriman sebagai "Hamba Allah" (anonim) berfungsi dengan benar. | | | | | |
| 5 | Proses unggah bukti pembayaran zakat berjalan lancar tanpa kendala teknis. | | | | | |
| 6 | Sistem memberikan notifikasi atau konfirmasi yang jelas setelah transaksi zakat diajukan. | | | | | |
| 7 | Fitur *Tracking* (pelacakan status zakat) memberikan informasi yang akurat (Pending/Sukses). | | | | | |
| 8 | Dashboard Muzakki menyajikan informasi riwayat transaksi yang rapi dan mudah dibaca. | | | | | |
| 9 | Tampilan sistem (UI) terlihat profesional, menarik, dan nyaman dipandang. | | | | | |
| 10 | Secara keseluruhan, sistem ini mempermudah saya dalam menyalurkan zakat ke LAZISMU. | | | | | |

---

### Lampiran: Skenario Pengujian (Untuk Penguji/User)
Untuk membantu responden menjawab pernyataan di atas, mintalah mereka melakukan skenario berikut:
1. **Pendaftaran:** Buatlah akun baru sebagai Muzakki.
2. **Transaksi:** Lakukan pengajuan zakat pada salah satu program, pilih opsi anonim, dan unggah foto bukti transfer.
3. **Monitoring:** Lihat halaman Dashboard dan Tracking untuk memastikan status pengajuan zakat Anda terlihat.
4. **Navigasi:** Cobalah berpindah-pindah menu untuk merasakan kecepatan dan desain antarmuka.
